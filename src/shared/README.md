# Shared Module

이 디렉토리는 **Main 프로세스**와 **Renderer 프로세스** 양쪽에서 공통으로 사용되는 타입 정의, 상수 및 유틸리티를 포함합니다.

## 디렉토리 구조

- **`refresh/`**: ChatGPT 대화 새로고침 및 데이터 갱신 관련 공통 타입
- **`sync/`**: 워크스페이스 스냅샷 및 클라우드 동기화(Google Drive) 관련 공통 타입
- **`import/`**: 프로젝트 단위 대화 가져오기 및 동기화 관련 공통 타입

## 설계 원칙
1. **프로세스 독립성**: 특정 프로세스(Electron/Browser) 전용 API를 직접 호출하지 않습니다.
2. **순수성**: 데이터 구조 정의와 순수 함수(Utility) 위주로 구성합니다.
3. **무결성**: 양측 프로세스 간의 통신 규격을 보장하는 단일 진실 공급원(Single Source of Truth) 역할을 합니다.
